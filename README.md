## 貼文版 TWB - 文章分享系統 Tieh-Wen-Ban

<img src="./TWBAndroid/screenshots/TWB.png" width="100%"/>

>手機論壇APP，介面類似Dcard，附上PTT的推與噓評價模式，圖片或文字皆可上傳。
### 特色
* App端為Android
* 套件使用 Glide, RxJava, okhttp, recyclerview....
* MVP架構
* HTTPs Request
